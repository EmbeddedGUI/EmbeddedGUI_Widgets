# AGENTS.md

This repository uses `CLAUDE.md` as the single source of truth for AI collaboration rules.

## Rules

1. All AI agents should follow `CLAUDE.md`.
2. `AGENTS.md` is only an entry point to avoid duplicated rules.
3. If `AGENTS.md` conflicts with `CLAUDE.md`, `CLAUDE.md` wins.
4. Update collaboration rules only in `CLAUDE.md`.
5. Always check `.claude/` first, especially `.claude/workflow/` and `.claude/skills/`, when the task touches widget workflow, runtime verification, or web publishing.
